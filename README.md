A Spring Boot application to manage internship postings — including title, location, company and stipend.

🚀 Features

	•	Add, view and filter through location, title or company
	•	REST API built with Spring Boot and Hibernate
	•	MySQL database integration

🏗️ Tech Stack

	•	Java 17
	•	Spring Boot 3
	•	Spring Data JPA (Hibernate)
	•	MySQL
	•	Maven
	•	REST API
